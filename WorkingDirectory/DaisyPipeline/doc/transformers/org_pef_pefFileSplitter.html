Transformer Purpose

The purpose of this transformer is to split a PEF file into several files, one file per volume.

Input Requirements

A PEF 2008-1 document.

Output

Multiple PEF 2008-1 documents.

On success

Multiple PEF 2008-1 documents.

On error
