Transformer Purpose

The purpose of this transformer is to convert a braille text file into a PEF 2008-1 document.

Note: This implementation is based on the December 9, 2009 Public Draft of the PEF 1.0 Specification.

Input Requirements

A braille text file.

Output

On success

A PEF 2008-1 document.

On error

Configuration/Customization

Parameters (tdf)

input
Path to input text file
output
Path to output PEF 2008-1 document
mode
Optional. Read mode

Optional. Document author
title
Optional. Document title
identifier
Optional. Document identifier
language
Optional. Document language
